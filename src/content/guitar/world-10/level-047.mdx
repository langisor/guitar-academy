# Level 47: Jazz Chords

## Lesson

Jazz chords add richness and sophistication to your playing by extending beyond the basic triads into 7th, 9th, and beyond. TheCmaj7 chord creates a dreamy, unresolved sound perfect for ballads and bossa nova. The Dm7 chord adds a slightly darker, more complex feel than a plain minor chord. The G7 chord acts as a powerful bridge that wants to resolve to C major. The Am7 is another versatile minor seventh chord that works in many contexts. These four chords form the foundation of jazz harmony in the key of C. Pay attention to the fingerings—the extra fingers needed for 7th and 9th chords require careful positioning to avoid muting strings.

## Exercise

Learn these essential jazz chord shapes step by step:
1. Start with Cmaj7: place your 1st finger on the 2nd string 1st fret, 2nd finger on 4th string 2nd fret, 3rd finger on 5th string 3rd fret
2. Strum all five strings (skip the low E)—check for clean sound
3. Practice switching between Cmajor and Cmaj7 to hear the difference
4. Now learn Dm7: 1st finger on 1st string 1st fret, 2nd finger on 2nd string 1st fret, 3rd finger on 3rd string 2nd fret
5. The Dm7 shape uses three fingers in a compact cluster—practice until clean
6. Learn G7: 1st finger on 1st string 1st fret, 2nd finger on 5th string 2nd fret, 3rd finger on 6th string 3rd fret, 4th finger on 3rd string 3rd fret
7. G7 is a four-finger chord—take your time building the shape
8. Learn Am7: 1st finger on 2nd string 1st fret, 2nd finger on 3rd string 1st fret, 3rd finger on 4th string 2nd fret
9. Practice the progression: Cmaj7 to Dm7 to G7 to Am7 and back
10. Work on smooth transitions, keeping the rhythm steady throughout
