# Level 3: Tuning Basics

## Lesson

Standard tuning is E-A-D-G-B-e. We use tuners to measure 'pitch.' If a note is too high, it's 'sharp'; too low is 'flat.'

## Exercise

Tune each string using a tuner.

## Tips

Aim for the green light on your tuner.
