# Level 4: String Names & Numbers

## Lesson

The six strings on a guitar are numbered from **1 to 6** when counting from the thinnest (highest-pitched) string to the thickest (lowest-pitched) string. The thinnest string is called the **high e** (or just "e" in lowercase), and the thickest is the **low E**. From lowest to highest, the string names are **E-A-D-G-B-e**. Use this helpful mnemonic: **"E**ddie **A**te **D**ynamite, **G**ood **B**ye **E**ddie." This catches every note in standard tuning. String numbers are crucial when following chord diagrams and tab notation, so memorize this order well. The numbering system is universal across all guitars.
